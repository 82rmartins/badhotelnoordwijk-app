**original_problem_statement:**
The user wants to build an internal executive mobile app for the Bad Hotel Noordwijk. The app's primary purpose is to provide a live operations display that allows a user to understand the hotel's status (under control or at risk) in 5 seconds.

**PRODUCT REQUIREMENTS (V1 - CSV First):**
- **Architecture:** A standalone Expo/React Native application with no backend dependency. All data processing and calculations must happen on the device.
- **Data Ingestion:** Data is loaded by an administrator via a CSV file upload screen within the app. The app must be prepared to parse CSVs from the Mews PMS.
- **Persistence:** Data should be stored on the device (e.g., using AsyncStorage) to allow for offline functionality, showing the last known data snapshot.
- **Core UI:**
    - An immersive, vertically scrollable feed-like interface.
    - **Fixed Header:** Displays the overall hotel status (🟢 Sob Controle / 🟡 Atenção / 🔴 Risco), a trend indicator (↑ Melhorando / → Estável / ↓ Piorando), a live clock, and a Last Update timestamp. A reason for the status should be shown if it's not Under Control.
    - **Section 1: TODAY (Operation):** A swipeable carousel showing key metrics (occupancy, arrivals/departures, revenue) for Yesterday, Today, and Tomorrow.
    - **Section 2: RADAR (14 Days):** A visual 14-day forecast of occupancy. It must highlight days that need attention.
    - **Section 3: CONTROL (Week & Month):** Contains swipeable carousels for:
        - A weekly/monthly occupancy chart.
        - Weekly performance stats (swipeable for 5 weeks: -2 past, current, +2 future).
        - Monthly performance stats (swipeable for 6 months: -2 past, current, +3 future).
- **Branding & Internationalization:**
    - Include the hotel logo.
    - The default language must be **English**.
    - A language toggle for **English (EN)** and **Dutch (NL)** must be present. All text must be internationalized.
- **Hotel Configuration:** The total number of rooms is fixed at **26**. All calculations must be based on this number.
